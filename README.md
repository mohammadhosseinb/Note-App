# 📝 Note App

A simple desktop note-taking application built with **Python** and **PySide6**.

## Features

- ➕ Add new notes
- 📋 Display notes in a list
- 👀 View note details
- ✏️ Edit notes (In Progress)
- 🗑️ Delete notes with confirmation
- 🆔 Unique ID for each note using UUID

## Technologies

- Python 3
- PySide6 (Qt for Python)
- UUID

## Installation

Clone the repository:

```bash
git clone git@github.com:mohammadhosseinb/Note-App.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python noteapp.py
```
